## Run Locally

### This app runs in node v16.13.2

Clone the project

```bash
  git clone https://github.com/ValEmpire/youset.git YouSet
```

Go to the project directory

```bash
  cd YouSet
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
